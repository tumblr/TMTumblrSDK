# PhotoPostExample

## Installation

Install `cocoapods-keys`.

https://github.com/orta/cocoapods-keys

```
$ gem install cocoapods-keys
```

Get `OAuth consumer Key` and `OAuth consumer Secret`.

https://www.tumblr.com/oauth/apps

Install libraries and set up keys.
Type your keys.

```
$ pod install
```
